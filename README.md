# Adela News Bot - Asistente Personal de Noticias con Inteligencia Artificial

## Descripción del Proyecto
Adela News Bot es una aplicación de escritorio desarrollada en Python que transforma la manera de consumir actualidad. El objetivo principal es filtrar el ruido informativo mediante el uso de modelos de lenguaje avanzados (GPT-4o-mini), permitiendo al usuario obtener resúmenes críticos y análisis de impacto en tiempo real.

Este proyecto nace de la necesidad de gestionar el exceso de información actual, priorizando la relevancia y la neutralidad en el contenido.

## Funcionalidades Principales

### 1. Análisis de Relevancia (Impact Scoring)
El sistema evalúa cada artículo mediante un proceso de análisis contextual, asignando una puntuación de impacto. Esto permite al usuario diferenciar rápido entre noticias de calado internacional y contenido de menor relevancia.

### 2. Modo de Debate Neutral
Una de las características diferenciales del bot es su capacidad para generar debates. Ante temas de actualidad complejos, el sistema es capaz de presentar dos perspectivas opuestas y equilibradas, fomentando un consumo de información crítico y libre de sesgos algorítmicos.

### 3. Navegación por Secciones Especializadas
La interfaz permite una segmentación por categorías (Economía, Política, Arte, Deportes, etc.), cada una con una configuración visual y lógica de búsqueda adaptada.

### 4. Accesibilidad e Interfaz de Voz
Implementación de un sistema de interacción híbrido que incluye:
- **Text-to-Speech (TTS):** Narración automatizada de resúmenes.
- **Speech Recognition:** Comandos de voz para el control de la aplicación, facilitando una experiencia manos libres.

## Stack Tecnológico
- **Lenguaje:** Python 3.13
- **IA y Procesamiento de Lenguaje:** OpenAI API (GPT-4o-mini)
- **Extracción de Datos:** NewsAPI (REST API)
- **Interfaz Gráfica:** Tkinter (Custom UI)
- **Gestión de Audio:** Pygame & gTTS
- **Entorno de Desarrollo:** Virtualenv para gestión de dependencias y limpieza del core de Python.
